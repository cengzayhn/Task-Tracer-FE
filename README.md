    <h1>TaskTracer</h1>

    <p>
        TaskTracer is a web application designed to help users plan and manage their daily tasks and duties more effectively. 
        The project focuses on enhancing users' work productivity, prioritizing tasks, and better organizing their daily lives.
    </p>

    <h2>Features</h2>
    <ul>
        <li><strong>Task Management:</strong> Easily create, edit, and delete tasks.</li>
        <li><strong>Prioritization:</strong> Set priority levels for tasks to focus on what matters most.</li>
        <li><strong>Daily Planning:</strong> Organize your day by scheduling tasks and setting reminders.</li>
        <li><strong>Productivity Insights:</strong> Track your progress and improve your work habits.</li>
    </ul>

    <h2>Installation</h2>
    <p>To get started with TaskTracer, follow these steps:</p>
    <ol>
        <li>
            <p><strong>Clone the repository:</strong></p>
            <pre><code>git clone https://github.com/cengzayhn/Task-Tracer-FE.git</code></pre>
        </li>
        <li>
            <p><strong>Navigate to the project directory:</strong></p>
            <pre><code>cd Task-Tracer-FE</code></pre>
        </li>
        <li>
            <p><strong>Install dependencies:</strong></p>
            <pre><code>npm install</code></pre>
        </li>
        <li>
            <p><strong>Start the development server:</strong></p>
            <pre><code>npm start</code></pre>
        </li>
    </ol>
